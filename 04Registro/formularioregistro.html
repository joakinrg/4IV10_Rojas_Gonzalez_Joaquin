GET:

Este metodo es el estandar como salida del navegador mediante el cual se resuelve una petición por parte del cliente, asi mismo en la salida estandar (url), se puede visualizar los datos enviados mediante el formualrio

POST:

Este metodo es utilizado para el envio de info dede un formulario atendiendo una peticion por parte del cliente actuando para poder agregar o insertar nuevos datos en una BD, asi mismoo a dif del metodo get, este utiliza "un tipo de seguridad" mediante el cual la info no es visible por medio de la salida estandasr del navegador.
